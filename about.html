项目简介

TechPulse 是一款面向手机相机工程师的自动化技术情报工具，每周自动抓取全球影像技术资讯，通过通义千问大模型从工程师视角提炼传感器规格、ISP 算法、计算摄影等专业技术要点，帮助工程师快速掌握行业动态，无需逐篇阅读原文。

🤖

AI 要点提炼

每条资讯自动读取原文网页，由通义千问（qwen-turbo）提炼技术要点，包含传感器尺寸、像素、光圈、变焦倍数、动态范围、ISP 型号、算法创新等具体参数。

⚙️

技术实现

  • Python 3.12 + feedparser 抓取 RSS/HTML
  • 多层关键词过滤排除无关内容
  • googlenewsdecoder 解析真实文章链接
  • GitHub Actions 每周自动运行并更新本站
